Le chat dort sur le canapé. Le chien regarde le chat dormir. Chaque matin, le chat se réveille avant le chien. Le chat saute du canapé, le chien le suit. Le chat mange, le chien mange aussi. Le chat aime le lait, le chien préfère l’eau. Le chat est calme, le chien est joueur. Le chat grimpe sur la table, le chien aboie. 

Dans le jardin, le chat court. Le chien court aussi. Le chat chasse un oiseau, le chien court derrière le chat. Le chat est rapide, le chien l’est un peu moins. Le chat s’arrête, le chien s’arrête. Le chat regarde, le chien regarde. Le chat s’assoit, le chien s’assoit.

Le soir, le chat retourne sur le canapé. Le chien s’allonge à côté du chat. Le chat ferme les yeux. Le chien ferme les yeux aussi. Le silence s’installe. Le chat respire lentement. Le chien ronfle un peu. Le chat bouge une oreille. Le chien ne bouge pas. Le chat dort. Le chien dort.
